# LuxeAds AI - Troubleshooting Guide\n\n## \"Generation failed\" Error - FIXED ✅\n\n### Root Cause\nThe app was calling the Anthropic API directly from the frontend, which:\n- Exposed your API key in browser network requests\n- Failed due to CORS restrictions\n- Had no error handling or logging\n\n### Solution\nAll requests now go through a **secure backend API** (`/api/generate`):\n\n```javascript\n// Frontend safely calls backend\nfetch('/api/generate', {\n  method: 'POST',\n  body: JSON.stringify({ projectData: form })\n})\n// Backend securely calls Claude with API key\n```\n\n---\n\n## Setup Instructions\n\n### 1.
